==== FILE: README.md ====
[1m[37m# everything[0m
[1m[37m(vibecoded)[0m
[1m[37m[0m
[1m[37mDump your entire project into a single file, mostly because you're about to feed it to an LLM, ran out of cursor credit, and can't be bothered to copy-paste 47 files.[0m
[1m[37m[0m
[1m[37mRecursively walks your directory and prints every file's path + contents. That's it. That's the tool.[0m
[1m[37m[0m
[37m```[0m
[1m[37meverything > context-for-llm.txt[0m
[37m```[0m[1m[37m[0m
[1m[37m[0m
[1m[37mWorks with any model. Works with any codebase. Zero dependencies (well, Go, but you already have that).[0m
[1m[37m[0m
[1m[37m---[0m
[1m[37m[0m
[1m[30m## Why this exists[0m
[1m[37m[0m
[1m[37mDon't know. Saves time everyone seemed to have anyway.[0m
[1m[37m[0m
[1m[37mStarted as a bash alias. Went skidding. Now it's a Go binary that still does the same thing but with flags.[0m
[1m[37m[0m
[1m[37mReal use cases people actually use this for:[0m
[1m[36m-[0m[1m[37m **Dumping code into LLM prompts** (the main one)[0m
[1m[36m-[0m[1m[37m Code review prep[0m
[1m[36m-[0m[1m[37m Quick project snapshots for sharing[0m
[1m[36m-[0m[1m[37m "Where did I put that function?" full-text search (pipe it into [0m[37m`grep`[0m[1m[37m — [0m[37m`everything | grep "func foo"`[0m[1m[37m)  <-- Not like it's built into ur IDE or anything[0m
[1m[37m[0m
